class ml\+::movar\+::filter$<$ Predicate $>$}
Forwards the argument if it satisfies a predicate, or returns \mbox{\hyperlink{structml_1_1movar_1_1nothing}{ml\+::movar\+::nothing}}. 

A filter is a lazy pipeline component that is constructed from a boolean predicate.

The predicate is invoked with the piped argument.

If the predicate returns true, the piped argument is forwarded to the next operation in the pipeline.

Otherwise, \mbox{\hyperlink{structml_1_1movar_1_1nothing}{ml\+::movar\+::nothing}} is returned.

See \mbox{\hyperlink{lazy-pipelines}{lazy-\/pipelines documentation}} for example usage.
